---

## ⚠️ Kenapa HARUS Manual (no automation)

| Reason | Explanation |
|--------|-------------|
| **CAPTCHA** | 70% of directories use reCAPTCHA/hCaptcha — automation tools get blocked |
| **Email verification** | Most require email click-through — needs real inbox access |
| **Account creation** | Each platform needs unique login (LinkedIn, Clutch, Trustpilot) |
| **Business verification** | Some require official docs (GBP needs postcard/postcard verification) |
| **Anti-spam detection** | Auto-submit triggers Google + platform spam filters |
| **Quality matters** | Google penalizes sites with low-quality directory spam patterns |
| **ToS violation** | Most directories explicitly prohibit automated submission → ban risk |
| **Unique data per platform** | Each has different form fields, categories, character limits |

**Conclusion**: Manual is 3-5 days work, automation is 1 day work + permanent ban. Choose manual.
