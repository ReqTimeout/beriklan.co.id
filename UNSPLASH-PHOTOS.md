# 📸 UNSPLASH PHOTO COLLECTION
## Beriklan.co.id Visual Library

> **Tujuan:** Replace real photo shoot dengan curated Unsplash collection dengan Indonesian-relevant representation + cool tone (match brand blue).
>
> **Cara pakai:**
> 1. Pilih foto dari section yang sesuai.
> 2. Download via URL dengan parameter `?w=2400&q=80&fm=avif` (atau format sesuai Astro `<Image>` config).
> 3. Apply color grade optional (subtle blue tint overlay).
> 4. Simpan di `apps/web/public/photos/{kategori}/{nama}.avif`.
> 5. Reference dari Sanity atau hardcoded paths.

---

## 🎯 Selection Criteria

**Wajib:**
- ✅ Indonesian / Asian representation (priority tinggi) ATAU universal professional dengan warm tone
- ✅ Modern work context (office, laptop, meeting, dashboard, mobile)
- ✅ Natural light (window / golden hour feel)
- ✅ Diverse composition (no staged group shots)
- ✅ Color palette friendly ke brand blue + indigo (cool tones)

**Hindari:**
- ❌ Handshake stock photos
- ❌ "Diverse group of people in business attire" yang unrealistic
- ❌ Oversaturated atau high-contrast
- ❌ Visible watermarks
- ❌ Heavy filter / terlalu stylized
- ❌ Identifiable people tanpa permission

## 🛠 Technical Specs

**Download format:**
```
https://images.unsplash.com/photo-{id}?w=2400&q=80&fm=avif&fit=crop
```

**Parameters:**
- `w`: width (2400 for hero, 1200 for thumbnails)
- `q`: quality 80 (good balance)
- `fm`: format (avif or webp)
- `fit`: crop or max

**Alt text template:**
```
[Deskripsi visual], [aksi/konteks], [mood].
+ Focus keyword if applicable untuk SEO.
```

**Contoh alt text yang baik:**
```
"Indonesian professional reviewing campaign dashboard on laptop in warm natural light, focused expression"
```

BUKAN:
```
"Image of business person on computer"
```

## 🎨 Color Treatment (Post-Processing)

Terapkan via CSS atau build-time processing:

```css
.unsplash-photo {
  filter: 
    sepia(8%)              /* warm tint */
    saturate(110%)         /* slight color boost */
    contrast(102%);        /* keep natural */
}

/* Brand overlay untuk konsistensi */
.unsplash-photo.brand-overlay::after {
  content: '';
  position: absolute;
  inset: 0;
  background: linear-gradient(
    135deg,
    rgba(37, 99, 235, 0.05) 0%,    /* blue subtle */
    rgba(255, 182, 39, 0.03) 100%   /* gold subtle */
  );
  mix-blend-mode: overlay;
  pointer-events: none;
}
```

## ✅ Final Checklist

Saat menggunakan foto di production:

- [ ] Download dalam format AVIF primary + WebP fallback
- [ ] Compress (target < 200KB per image)
- [ ] Apply color treatment (subtle blue tint overlay)
- [ ] Tulis alt text (100-250 char, descriptive, include keyword kalau applicable)
- [ ] Credit Unsplash + photographer di page tertentu (jika required)
- [ ] Set lazy loading kecuali hero (eager + fetchpriority="high")
- [ ] Set width + height attribute (prevent CLS)
- [ ] Generate responsive srcset untuk breakpoints
